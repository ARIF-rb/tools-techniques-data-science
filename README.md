# Tools and Techniques of Data Science

Course work and assignments for the **Tools and Techniques of Data Science** course. Covers web scraping, data pipeline design with KNIME, and full-stack portfolio web development.

## Contents

| Folder | Description |
|--------|-------------|
| [faculty-web-scraper](./faculty-web-scraper) | Assignment 1 — Web scraper extracting faculty profiles from a university website |
| [knime-retail-pipeline](./knime-retail-pipeline) | Assignment 3 — Retail data processing and visualization pipeline built in KNIME |
| [flask-portfolio-website](./flask-portfolio-website) | Assignment 4 — Personal portfolio website built with Flask and Bootstrap (Meyawo theme) |

## Tech Stack

- Python (requests, BeautifulSoup, pandas)
- KNIME Analytics Platform
- Flask, HTML/CSS/SCSS, Bootstrap 4, Gulp
