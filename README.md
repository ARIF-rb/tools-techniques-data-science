# Tools and Techniques of Data Science

Course work and assignments for the **Tools and Techniques of Data Science** course. Covers web scraping, no-code data pipelines with KNIME, and full-stack portfolio web development.

## Projects

| Folder | Type | Description |
|---|---|---|
| [faculty-web-scraper](./faculty-web-scraper) | Assignment 1 | Web scraper extracting faculty profiles from a university website into structured CSV |
| [knime-retail-pipeline](./knime-retail-pipeline) | Assignment 3 | Retail data processing, cleaning, and visualization pipeline built in KNIME Analytics Platform |
| [flask-portfolio-website](./flask-portfolio-website) | Assignment 4 | Personal portfolio website built with Flask and Bootstrap 4 (Meyawo theme) |

## Tech Stack

- Python (requests, BeautifulSoup, pandas)
- KNIME Analytics Platform
- Flask, HTML/CSS/SCSS, Bootstrap 4, Gulp

## How to Navigate

Each subfolder is a standalone assignment with its own README, setup instructions, and usage notes. Click a folder link above to get started.
